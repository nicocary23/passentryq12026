🔮 The Big One: Secure Element

We are building the capability to issue proprietary payment credentials directly into the iPhone and Android using Secure Element — the same hardware chip that powers Apple Pay and Google Pay. Apple opened third-party access in 2024. We are moving into that gap. Face ID, tap, instant payment, zero interchange. Our first potential client is Frasers Group.

This is not a feature update. This is PassEntry moving up the stack from wallet pass infrastructure to embedded payments infrastructure.
